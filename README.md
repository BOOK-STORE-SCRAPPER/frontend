# Frontend

Next.js app for the Book Store Scraper.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Scrape categories from the backend.
- Select categories to scrape books.
- Display scraped data.
